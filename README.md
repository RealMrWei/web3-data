# Web3 Data Parser
EVM blockchain on-chain data collection and analysis tool.

## Features
- Fetch block and transaction data via RPC
- Parse ERC20 / NFT transfer events
- On-chain log decoding and data cleaning
- Support Ethereum, BSC, Polygon

## Tech Stack
Java | Web3j | RPC API | Blockchain Data Processing

## Use Case
Suitable for blockchain data backend, DApp data service and on-chain analytics.
